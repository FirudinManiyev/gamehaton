# .DOT GAMES 🎮

## Problem

Müasir oyun platformaları istifadəçilərə minlərlə oyun təqdim edir. Steam, Epic Games və Xbox Game Pass kimi xidmətlərdə böyük oyun kitabxanasına sahib olmaq üstünlük kimi görünsə də, bu vəziyyət yeni problemlər yaradır.

### Doğru Oyunu Tapmaq Çətindir

İstifadəçilər hər gün yüzlərlə yeni oyun, endirim və tövsiyə ilə qarşılaşırlar. Lakin bu qədər seçim arasında həqiqətən maraqlarına, oyun zövqlərinə və vaxtlarına uyğun oyunu tapmaq çətinləşir.

Nəticədə:

- İstifadəçilər seçim etməkdə çətinlik çəkirlər
- Uzun müddət oyun seçmək üçün vaxt sərf olunur
- Maraqlı oyunlar diqqətdən kənarda qalır
- İstifadəçi təcrübəsi zəifləyir

### Oyun Kitabxanasından Effektiv İstifadə Olunmur

Steam, Epic Games və Game Pass kitabxanalarında yüzlərlə oyun toplansa da, onların böyük hissəsi heç oynanmır və ya yarımçıq qalır.

Nəticədə:

- Yeni oyunlar alınır, köhnələr isə unudulur
- İstifadəçilər sahib olduqları oyunların fərqində olmurlar
- Oyunlara xərclənən vəsaitin səmərəliliyi azalır
- Kitabxanada "nə oynayım?" problemi yaranır

---

## Həll

**.DOT GAMES** istifadəçilərə sahib olduqları oyunlardan maksimum faydalanmağa və öz zövqlərinə uyğun oyunları daha rahat kəşf etməyə kömək edən platformadır.

Platforma istifadəçinin oyun kitabxanasını analiz edir, oynama vərdişlərini nəzərə alır və fərdiləşdirilmiş tövsiyələr təqdim edir.

---

## Əsas Funksiyalar

### Smart Game Discovery

- İstifadəçi maraqlarına uyğun oyun tövsiyələri
- Janr, reytinq və oyun müddətinə əsaslanan filtrasiya
- Yeni kəşf edilə biləcək oyunların təqdim edilməsi
- Fərdiləşdirilmiş oyun siyahıları

### Library Analytics

- Steam, Epic Games və Game Pass kitabxanalarının bir yerdə görüntülənməsi
- Ən çox oynanan oyunlar
- Uzun müddətdir oynanmayan oyunlar
- Tamamlanmamış oyunların siyahısı
- Oyun statistikaları

### Smart Recommendation Engine

- Oyun tarixçəsinə əsaslanan tövsiyələr
- Gizli qalmış və oynanmayan oyunların aşkarlanması
- "What Should I Play Today?" funksiyası
- İstifadəçinin maraqlarına uyğun seçimlər

### Personal Dashboard

- Kitabxananın ümumi statistikası
- Oyun fəaliyyəti analizi
- Tövsiyə olunan oyunlar
- Oyun məqsədləri və progress izləmə
